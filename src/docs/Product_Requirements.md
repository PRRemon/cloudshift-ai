# CloudShift AI - Product Requirements

## Vision

CloudShift AI is an enterprise SaaS platform that helps organizations assess, plan, and manage migration from on-premises infrastructure to AWS.

---

## Problem Statement

Many organizations still maintain migration plans using spreadsheets and manual documentation. This makes dependency tracking, migration planning, and cost estimation difficult.

CloudShift AI centralizes migration planning into a single platform.

---

## Target Users

* Cloud Architects
* Infrastructure Engineers
* Project Managers
* DevOps Engineers
* IT Managers

---

## Goals

* Centralize migration planning.
* Analyze infrastructure inventory.
* Recommend AWS target services.
* Estimate migration costs.
* Track migration progress.
* Provide AI-powered migration assistance.

---

## Out of Scope (MVP)

* Live server discovery
* Real AWS account integration
* Automatic infrastructure provisioning
* Billing integration
